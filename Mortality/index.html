Moratility Plugin

Mortality is a feature-rich PvP plugin for Paper 1.21 combining a unique heart system, combat-log protection, teams, and a mention system.



The plugin is still in development.

Download

Features

Heart System

Every player starts with 3 lives. Dying in combat costs one heart. Reaching 0 hearts kicks the player from the server. Admins can adjust hearts with /sethearts.

Combat System

Hitting another player starts a 30-second combat timer, extended by further damage. A boss bar displays the remaining combat time at all times.

Combat-Log Protection

If a player disconnects during combat, a Mannequin NPC automatically spawns in their place, wearing their armor and holding their items. Any damage dealt to the NPC is applied to the player upon rejoin.

Team System

Create teams with a custom name, display name, and color. Invite players, chat privately via /tc or the # prefix, and manage everything through /team.

Mention System

Typing an online player's name in chat highlights it in yellow and sends them a sound notification. The sound can be toggled per-player via /settings.

Commands

Command Description Permission
/lang <de|en> Set your preferred language
/sethearts <player> <amount(1-3)> Set the number of hearts for a player heartsystem.set
/settings Open the settings menu
